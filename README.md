<h1 align="center">
  @pansy/china-division
</h1>

> 提供中国省市区数据，

**提供两种数据格式**

- `{ v: string; n: string; c: {...}[] }`
- `{ label: string; value: string; children: {...}[] }` 适用于`Antd`级联选择组件

## 🏗 安装

```sh
# yarn
yarn add @pansy/china-division

# npm
npm install @pansy/china-division --save 
```

## 🔨 使用

```ts
import pcaData from '@pansy/china-division';

// you code
```
